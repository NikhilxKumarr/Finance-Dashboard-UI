# 💰 Finance Dashboard UI (₹)

A modern, responsive financial dashboard built using **React.js** and **Tailwind CSS**, designed to track income, expenses, and financial insights with an intuitive user interface.

---

## 🚀 Live Demo

👉 https://finance-dashboard-ui-woad.vercel.app/ 

---

## 📌 Features

* 📊 **Dashboard Overview**

  * Summary cards for Balance, Income, and Expenses
* 📈 **Time-Based Visualization**

  * Balance trend using line charts
* 🧾 **Transaction Management**

  * Add, edit, and delete transactions
* 🔍 **Search & Filtering**

  * Filter by category, type, and search keywords
* 📊 **Categorical Insights**

  * Spending breakdown using charts
* 👥 **Role-Based UI**

  * Viewer and Admin modes
* 🌙 **Dark Mode**

  * Toggle between light and dark themes
* 💾 **Local Storage Persistence**

  * Data saved across sessions
* 📱 **Responsive Design**

  * Works on desktop, tablet, and mobile

---

## 🛠️ Tech Stack

* **Frontend:** React.js (JavaScript)
* **Styling:** Tailwind CSS
* **Charts:** Recharts
* **Icons:** Lucide React
* **State Management:** React Context API

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/NikhilxKumarr/Finance-Dashboard-UI.git
cd Finance-Dashboard-UI
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm start
```

### 4. Open in browser

```
http://localhost:3000
```

## ⚖️ Technical Decisions

* Used **React Context API** for lightweight state management
* Chose **Tailwind CSS** for fast and consistent UI development
* Used **Recharts** for simple and interactive data visualization
* Stored data in **localStorage** to avoid backend complexity

---

## ⚠️ Limitations

* No backend (data stored locally)
* No authentication system
* Role-based access is simulated (frontend only)
